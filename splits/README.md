# Dataset Splits

This directory is reserved for the dataset split files used in the paper. No split files are included in the pre-acceptance preview.

After acceptance, release split files should be placed under:

```text
splits/whu-stereo/
splits/us3d/
```

Recommended format: plain text files with one sample identifier per line, such as `train.txt`, `val.txt`, and `test.txt`. JSON or CSV metadata files may be added if needed.
